# Security policy

## Status

supernova is in development and has no tagged release yet; do not deploy it in
production. Security fixes will target the latest release once one exists.

## Reporting a vulnerability

Please report suspected vulnerabilities privately via GitHub's
[private vulnerability reporting](https://github.com/quasarnova-team/supernova/security/advisories/new)
("Security" tab → "Report a vulnerability"). Do not open a public issue.

We aim to acknowledge reports within **72 hours**. Issues inherited from the upstream
quasar framework will be coordinated with the upstream project.
